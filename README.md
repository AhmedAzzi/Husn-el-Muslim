## 🌟 Features

- 📖 **Comprehensive Adkar**: All the essential supplications from "Husn El Muslim".
- 🎧 **Audio Playback**: Professional audio recitations for all Adkar with an integrated media player.
- 🔍 **Enhanced Search**: Find Adkar instantly with an optimized search bar (autofocus, clear button).
- 🎨 **Modern UI**: Clean, premium design with support for **Dark Mode** & **Light Mode**.
- 📤 **Smart Sharing**: Share Adkar as text or beautiful screenshots directly to social media.
- 🧮 **Digital Tasbih**: Integrated counter for Adkar with visual feedback.
- 🌍 **RTL Support**: Native design for Arabic language and right-to-left layout.
- 📋 **Copy to Clipboard**: One-tap copy for sharing text externally.

- **[Flutter](https://flutter.dev/)** - UI Framework.

- **[share_plus](https://pub.dev/packages/share_plus)** - Cross-platform sharing.

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (latest stable recommended)
- Android Studio / VS Code
- Android/iOS Emulator or Physical Device

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AhmedAzzi/Husn-el-Muslim.git
   cd husn_el_muslim
   ```

2. **Install dependencies:**
   ```bash
   flutter pub get
   ```

3. **Run the app:**
   ```bash
   flutter run
   ```
